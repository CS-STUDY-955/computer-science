## :loudspeaker: Rule
1. 요일 : 화, 금
2. 주제 : 화 - 알고리즘 제외 대주제 분배 후 소주제 자유 선택
3. 주제 선정
   - Computer Science: 화 PT 후 차주 대주제 분배
   - Class Review: 금 PT 후 차주 커리큘럼에 있는 주제 분배
4. 시간 : PT 10분 / 질문 5분
